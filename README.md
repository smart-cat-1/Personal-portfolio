# Personal Portfolio

A personal portfolio website built by HTML + CSS + JavaScript.

## Project Features

- Multi-page structure（Navigation / Information / Education / Projects / Work Experience）
- Dark/Light Theme Toggle
- Home Page Interactive Star Menu
- Like/Dislike/Love Button Animation
- Responsive layout style
- Pure frontend static deployment, ready to use out of the box
